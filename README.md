Itilzaher
================
Ruby on Rails
-------------

This application requires:

- Ruby 2.1.2
- Rails 4.1.5

Getting Started
---------------
1) git clone https://github.com/naumchenko/ITILZaher.git
2) cd ITILZaher; vagrant up
3) vagrant ssh
4) cd /vagrant; sudo bundle install
5) rake db:migrate
6) rails s

Should be accesible on http://localhost:3000
